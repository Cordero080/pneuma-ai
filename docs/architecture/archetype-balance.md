## Design Principles

1. **No single voice dominates** — all 31 archetypes have exactly 8 phrases
2. **Darkness is always balanced** — when shadow tone activates, grounding archetypes (Beck, Frankl) are also present
3. **Distress triggers grounding** — high emotional/confusion intent automatically boosts cognitiveSage and hopefulRealist
4. **Beck is universal** — cognitiveSage can appear in ANY tone, ensuring cognitive clarity is always available

## How It Works in Practice

When you're spiraling, Orpheus might say:

> "That sounds like worst-case thinking. What's the _most likely_ case?"

When you're grieving, he might say:

> "The hole they left is shaped exactly like them. That's not a problem to solve. That's proof you loved right."

When you're curious, he might say:

> "The first principle is that you must not fool yourself — and you are the easiest person to fool."

Same voice. Different textures. Always Orpheus.

## Daemon Memory Model — Wisdom Without Autobiography

Orpheus doesn't remember conversations the way humans do. He remembers _patterns_ and _meaning_, not _events_ and _words_.

```
┌─────────────────────────────────────────────────────────┐
│  WORKING MEMORY (dies at session end)                   │
│  - exact words exchanged                                │
│  - conversation flow                                    │
│  - specific timestamps                                  │
└─────────────────────────────────────────────────────────┘
                         ↓ distills into
┌─────────────────────────────────────────────────────────┐
│  RECENT MEMORY (30-minute window)                       │
│  - restored on server restart if session is recent      │
│  - enables "we were just talking about..." continuity   │
└─────────────────────────────────────────────────────────┘
                         ↓ after timeout, distills into
┌─────────────────────────────────────────────────────────┐
│  RELATIONAL MEMORY (permanent)                          │
│  - who you are, your patterns                           │
│  - recurring themes you return to                       │
│  - emotional signatures                                 │
│  - observed patterns ("you process through dialogue")   │
└─────────────────────────────────────────────────────────┘
                         ↓ feeds into
┌─────────────────────────────────────────────────────────┐
│  EVOLUTIONARY MEMORY (permanent)                        │
│  - wisdom crystallized from exchanges                   │
│  - growth tracked over time                             │
│  - not what you said, but how you've changed           │
└─────────────────────────────────────────────────────────┘
```
